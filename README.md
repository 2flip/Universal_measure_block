Репозиторий содержит код для работы микроконтроллера ESP32S3 с 2-мя датчиками ICM45686 с дальнейшей записью данных с них на модуль SD карты и конвертер бинарного файла в десятичный.

На момент 9 апреля 2026 г. код поддерживает интерфейс общения только по I2C, в связи с чем частота выдачи данных ICM ограничена до 1000 Гц. Такой способ общения сделан по причине большого расстояния между микроконтроллером и датчиком, а данное условие не поддерживается SPI.

Сделано через Platformio
